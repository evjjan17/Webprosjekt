**Allow users to draw custom routes and generate a catalog of map routes and trails with points of interest on a Wordpress site using Google Maps integration.**

The Google Maps Routes Manager allows users to draw custom routes and generate a catalog of map routes and trails with points of interest (locations). With Google Maps integration, users can display images, descriptions, and searchable markers with each location including elevation and temperature. 

This Google Maps plugin lets users choose route direction, localize routes, add geographical tags, and fully customize maps and points of interest within WordPress.

[vimeo https://vimeo.com/144472757]

The Google Maps Routes Manager lets users include customized walking paths, car routes, motor bike trails, bicycle routes and more.

Get support from Google Maps to import and export map routes for navigation on mobile devices. This Google maps plugin will automatically calculate the best route based on route direction.

 == Basic Location Plugin Features ==

* Draw routes on Google map
* Add description images for each route
* Show all routes on an index page

== Benefits of Going Pro ==

* Upload route from a KML/GPX files – Routes can be imported from a device that generates a KML/GPX files. Mobile phones and navigation devices can generate such files.
* Export route to a KML/GPX files – Routes created can later be exported to a KML/GPX files and shared with mobile devices, other travel sites, and navigation aids.
* Weather information – Weather information is shown near each location.
* Altitude graph – Altitude is added per each trail view and is also accompanied by information such as trail distance, average speed, peak altitude, and more.
* Images and video – Images and video can be added to both trail information and each location
* Categories – Routes can be assigned to categories and filtered by categories
* Shortcodes – Several shortcodes are supported. Shortcodes can be embedded in posts and show a single location, a map with all locations in a category, and more.
* Tags – Tags can be added to routes and allow filtering of routes
* Search – Trail description, locations, and route name can be searched by keywords.
* Custom Icons – Upload your own custom icons or user any icon from available custom icons.
* Integrate with CM Map Locations to show both routes and locations on same map.
* Additional taxonomies for route difficulty and grade
* Many more options to customize view

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add a Google API key.
4. Add as many routes as you want. Each will have it's own route page and will show up on the index map
5. There are a handful of other optional preferences available in the dashboard.
